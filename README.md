### Usage

.env file should look like this:

    REACT_APP_BASE_URL = databaseName
    REACT_APP_API_KEY  = API_KEY

### Getting started

    git clone https://github.com/merima98/BurgerBuilder.git
    npm install
    npm start

### Testing application:

    npm test
